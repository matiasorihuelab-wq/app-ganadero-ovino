# Excel de referencia

Acá va el **Excel definitivo** (la fuente de verdad del modelo) cuando esté disponible,
para la auditoría de fidelidad.

- Guardar el archivo en esta carpeta (p. ej. `modelo-ovino.xlsx`).
- Anotar abajo su versión/fecha para trazabilidad.
- El repositorio es **privado**: el Excel puede versionarse acá sin riesgo de exposición.

> Todavía **no incorporado**. Ver el proceso en [../README.md](../README.md).
