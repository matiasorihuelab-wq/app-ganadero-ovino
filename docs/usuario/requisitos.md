# Requisitos mínimos

La aplicación corre **dentro del navegador**. No instala programas ni requiere
permisos especiales.

## Para la versión "archivo único" (la más simple)

- Una **computadora** (Windows, Mac o Linux) o un **celular/tablet** moderno.
- Un **navegador actualizado**: Chrome, Edge, Firefox o Safari (de los últimos 2-3 años).
- **No necesita internet** una vez que tenés el archivo.

## Para la versión Web / PWA

- Lo mismo que arriba, **con conexión a internet** para la primera carga.
- La dirección (URL) donde esté publicada la aplicación.

## Recomendado

- Pantalla de al menos 1024 px de ancho para ver el formulario y los resultados lado a
  lado (en celular se apilan, también funciona).
- Tener el navegador con **JavaScript activado** (viene activado por defecto).

## No hace falta

- No hace falta instalar nada, ni crear una cuenta, ni conectarse a un servidor.
- No hace falta Excel ni ningún otro programa.
