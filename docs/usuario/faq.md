# Preguntas frecuentes (FAQ)

**¿Necesito internet?**
Para la versión de **archivo único**, no. Para la versión **web**, solo para la primera
carga (después funciona offline si la instalaste como app).

**¿Dónde se guardan mis datos? ¿Los ve alguien?**
Se guardan **solo en tu navegador**, en tu propia computadora. **No se envían a ningún
servidor** ni los ve nadie más.

**¿Pierdo mis datos si cierro la aplicación?**
No: el trabajo en curso se **autoguarda** y los escenarios que guardás con nombre quedan
disponibles. Pero ojo: si **borrás los datos de navegación**, usás **otro navegador** u
**otra computadora**, esos datos no aparecen (no se sincronizan). Ver
[limitaciones](limitaciones.md).

**¿Puedo usarla en el celular?**
Sí. Se adapta a la pantalla (el formulario y los resultados se apilan).

**¿Sirve para cualquier raza?**
Sí, es un **template genérico**: todos los campos arrancan en 0 y cargás los tuyos. El
botón "Cargar ejemplo" usa datos de un Merino solo a modo de demostración.

**Los porcentajes, ¿cómo se cargan?**
Como número entero o decimal (ej. **80** para 80 %). La app los maneja internamente.

**¿Por qué algún resultado da 0 o no aparece?**
Probablemente falte un dato clave (ovejas, precios, micronaje, superficie). Mirá los
**avisos** que aparecen arriba de los resultados.

**¿Cómo saco un PDF?**
Botón **📥 PDF** → se abre el diálogo de impresión → elegí **"Guardar como PDF"**.

**¿Los números son definitivos?**
Esta es una versión **RC2**: el motor de cálculo todavía se va a **validar contra el
Excel** de referencia. Algunos valores podrían ajustarse. Ver
[limitaciones](limitaciones.md).

**¿Cómo comparto la aplicación con otro productor?**
Pasale el archivo **`app-rentabilidad-ovina.html`** (versión de archivo único). Lo abre
con doble clic, sin instalar nada.

**Encontré un error o algo raro. ¿Qué hago?**
Ver [Cómo reportar errores](reportar-errores.md).
